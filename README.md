# options-learning
This is my “Options Learning” repository, used for:  
- Storing learning plans and notes  
- Demonstrating models such as Black–Scholes  
- Managing experimental code and data  

## Directory Structure
- `docs/`: Documentation and progress  
- `notebooks/`: Jupyter notebooks  
- `src/`: Modular scripts  
- `data/`: Sample data  
- `tests/`: Unit tests  

## Quick Start
1. Clone the repository  
2. Install dependencies: `pip install -r requirements.txt`  
3. Explore the `notebooks/` directory to run experiments
